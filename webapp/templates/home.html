Services:

  • Troubleshooting

    • Computers fail and often need routine maintenance to keep them working optimally. I am able to remove malware, clean up files on your hard drive, and other maintenance. If you have a problem, e.g., your internet connection is not working, I can work to resolve the issue.

  • New device setup

    • If you have a new computer, wifi router, printer, speakers, I can help get it up and running. I can download the necessary software and configure the settings of your new PC or Mac to connect to the internet, be able to send emails, be protected from malware, etc. If you want to get a new computer or other device but are not sure which one to choose, I can help you find the one that works best for you.

  • Scripting automated tasks

    • If you have any menial computer tasks that need to be performed often, or take lots of time, I can help. Python is one of the best scripting languages for automating these tasks, and with the wide array of Python libraries for manipulating excel spreadsheets or web scraping, these tasks can be performed in mere seconds. If you need to I can create a script you can run on your own computer.

  • Tutorials of applications

    • If you do not know how to use certain applications on your computer such as Google Docs, Microsoft Word, or Gmail, I am happy to show you the basics and customize the settings.

  • Backup files to the cloud

    • Most people recommend having another copy of irreplaceable files (your pictures, important word documents, etc.,)
    Computers hard drives can and do fail, and having a copy of your files in the cloud, where companies such as Microsoft, Google, and Amazon can make sure they are safe, is becoming more and more popular. With the popular photo backup website for example, Google Photos, you can access your photos on any computer you sign in on. I can also set it up to back-up all files from the folder, making it simple for you to safeguard new photos by dragging and dropping.
    I am happy to help you secure your PC, word docs, photos.

  • Password Manager

    • If you, like everyone eles, struggle with remembering all your passwords, there is an easy fix; password managers. Password managers allow you to enter passwords (keeping them stored safely in your account in the cloud) that are filled in automatically in your web browser. I can set up a password manager for your computer and show you how to use it.
